## Piscine Python for Data Science (42)

Piscine intensive dédiée à l’utilisation de Python pour la manipulation, l’analyse et la visualisation de données.

### Contenu

- Manipulation de données avec Python
- Analyse et traitement de datasets
- Visualisation de données
- Résolution d’exercices progressifs

### Contexte

Piscine de l’école 42 orientée vers la découverte des bases de la data science et la prise en main des outils Python associés.
